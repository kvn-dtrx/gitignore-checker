# Gitignore Checker

## Synopsis

To emulate gitignore mechanisms.

A welcome opportunity to program in Rust.

## Installation

clone and enter

``` sh
git clone https://github.com/kvn-dtrx/gitignore-checker.git &&
    cd gitignore-checker
```

Build project

``` sh
cargo build
```

## Usage

``` sh
cargo run --release -- arg1 arg2 --flag
```
